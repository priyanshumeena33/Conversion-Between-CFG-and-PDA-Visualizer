# Conversion-Between-CFG-and-PDA-Visualizer
🚀 CFG ↔ PDA Visualizer This Web App demonstrates the conversion between Context-Free Grammars and Pushdown Automata. It features seamless dual-way conversion, step-by-step logic derivation, and dynamic graphviz state diagrams. Built with Tailwind CSS and Vanilla JS, it’s a pro tool for visualizing Automata Theory and the Equivalence Theorem.
